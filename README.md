# minor_project_1
GroupDNA is a Python data analytics tool that turns raw WhatsApp chat exports into interactive visual insights. It decodes your chat data to reveal hidden trends, from peak messaging hours to your group's most-used emojis.
The main libraries and modules utilized in your **GroupDNA** project include:

* **Pandas & NumPy:** Used extensively for structural data cleaning, handling complex datetime objects, and fast data manipulation.
* **Regex (`re`):** Essential for complex string parsing to separate raw WhatsApp chat text, timestamps, and authors from the system messages.
* **Matplotlib & Seaborn:** Used to build visual elements such as active user charts and activity heatmaps.
* **WordCloud & Emoji Libraries:** Leveraged for text mining, isolating vocabulary trends, and tracking the group's specific emoji signatures.
